📱 Projeto de Criação de Site — Curso Mobile
📖 Sobre o Projeto
Este projeto tem como objetivo desenvolver um site moderno, responsivo e funcional como parte das atividades do curso de Mobile. O foco é aplicar conceitos de desenvolvimento web, design responsivo e integração entre tecnologias utilizadas no ambiente mobile e web.
O site poderá ser utilizado para apresentar informações, produtos, serviços ou conteúdos relacionados ao tema escolhido durante o curso.

🚀 Tecnologias Utilizadas


HTML5


CSS3


JavaScript


Bootstrap


Git e GitHub



🎯 Objetivos do Projeto


Criar uma interface responsiva


Desenvolver páginas modernas e intuitivas


Aplicar conceitos de UX/UI


Organizar estrutura de arquivos do projeto


Utilizar versionamento com Git

⚙️ Como Executar o Projeto


Clone este repositório:


git clone https://github.com/seuusuario/projeto-site-mobile.git


Acesse a pasta do projeto:


cd projeto-site-mobile


Abra o arquivo index.html no navegador.
